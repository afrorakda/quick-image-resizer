# #34 Quick Image Resizer

A high-performance, browser-native tool to resize images instantly while maintaining the original aspect ratio.

## Core Philosophy
* **1 Tool / 1 Action**: Focuses solely on resizing images to a specific width without unnecessary editing features.
* **Instant Preview**: The drop zone transforms into a preview area as soon as an image is loaded, giving you immediate visual feedback.
* **Privacy First**: All processing happens locally via the Canvas API. Your images are never uploaded to any server.

## How to Use
1. **Drop an image** (JPG, PNG, or WebP) into the dashed box or click to select.
2. The tool automatically detects the original width. **Enter your target width** in the input field.
3. The aspect ratio is maintained automatically.
4. Click **Resize & Download** to save your optimized image.

## Features
* **Subtitle Guide**: "Resize images in your browser instantly. No upload, no tracking."
* **Smart UI**: The drop zone doubles as a preview window once an image is selected, keeping the interface compact.
* **Optimized for Mobile**: Features 15px rounded corners and large, easy-to-tap buttons for on-the-go adjustments.

## Privacy & Performance
* No login / No tracking / No ads / Browser-only.
* Zero server latency—works at the speed of your device's hardware.
